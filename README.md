## Description

Authentication via Discord using OAuth2 to an existing app.

Main reference :
https://www.nerd.vision/post/nestjs-third-party-oauth2-authentication

Don't forget to create your discord API token + paste those credentials in the strategy

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
